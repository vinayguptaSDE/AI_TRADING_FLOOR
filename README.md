# AI Trading Floor
Build a Trading Floor with 4 agents making autonomous trades, powered by 6 MCP Servers and 44 tools

## Features
- Multiple autonomous trading agents
- Strategy execution using MCP servers
- Real-time market simulation

## Tech Stack
- Python
- MCP (Model Context Protocol)
- AI Agents

## Setup

1. Clone the repo:
git clone https://github.com/vinayguptaSDE/AI_TRADING_FLOOR.git

2. Install dependencies:
pip install -r requirements.txt

3. Run:
python app.py

## Note
This project may require Python 3.7 and older dependencies (TensorFlow 1.x).
